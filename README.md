# منصة توليد تقارير الهندسة القيمية | Value Engineering Report Generator

منصة متخصصة لإعداد تقارير دراسة الهندسة القيمية وفق النموذج المعتمد من هيئة كفاءة الإنفاق والمشروعات الحكومية (EXPRO).

## النشر على Vercel

### الطريقة 1: رفع المجلد مباشرة
1. اذهب إلى [vercel.com](https://vercel.com)
2. اضغط **Add New** → **Project**
3. اختر **Import** من Git أو ارفع المجلد
4. سينشر تلقائياً

### الطريقة 2: عبر GitHub
1. أنشئ مستودع جديد على GitHub
2. ارفع جميع الملفات
3. اربط المستودع بـ Vercel

## الملفات
```
├── public/
│   └── index.html      ← المنصة الكاملة
├── vercel.json          ← إعدادات Vercel
├── package.json
└── README.md
```
